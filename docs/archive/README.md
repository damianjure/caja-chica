# docs/archive

Specs y planes **históricos** escritos a mano, **superados por OpenSpec**.

> La fuente de verdad para specs/cambios es **`openspec/`** (`openspec/changes/` para
> cambios en curso/archivados, `openspec/specs/` para specs canónicas).
> Lo de acá se conserva solo como referencia de cómo se documentaba antes.

- `specs/` — specs/planes sueltos previos a OpenSpec.
- `superpowers/` — planes generados por el workflow superpowers.
